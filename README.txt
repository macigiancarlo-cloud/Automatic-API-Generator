# AUTOMATIC REST API GENERATOR - DEMO VERSION

INSTRUCTIONS FOR USE:

1. Install Python on your computer.
2. Install the required libraries:
   Open the terminal in the folder and type: pip install -r requirements.txt
3. Place your SQLite database in the folder and name it 'database.db'.
4. Launch the interface:
   Type in the terminal: python app_dashboard.py
5. Open the browser at: http://127.0.0.1:5001

DEMO LIMITATIONS:
- This version generates code only for the FIRST table found in the database.
- Other tables will be ignored.
- The generated code contains a promotional watermark.
